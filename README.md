Global Events, Commodities and Currency Markets Analysis

This project was completed as part of the Applied Data Analytics and Market Research Assignment. The purpose of this analysis was to explore historical trends in Brent Oil, Gold and Silver markets and understand how commodity prices change over time.

The project focuses on analysing commodity market behaviour using historical datasets. Different analytical techniques were used to understand price movements, identify important trends and compare commodity behaviour during major global events and periods of uncertainty.

The datasets used in this project include Brent Oil, Gold and Silver historical price data. These datasets were analysed using Python and Jupyter Notebook to generate insights, visualisations and summary tables.

The main objective of this work was to identify historical price trends, understand periods of high and low market activity and observe how commodities respond during periods of economic uncertainty or market disruption.

The questions selected for this assignment include identifying the date range available in the datasets, analysing the highest and lowest Brent Oil prices, identifying the top-performing Brent Oil years, visualising Brent Oil price trends and comparing the latest Gold value with its long-term historical average. Additional analysis was performed using moving averages and a simple trendline forecast to better understand long-term market direction.

The project was completed using Python along with libraries such as pandas, NumPy and matplotlib for data cleaning, analysis and visualisation.

The project folders are organised to keep the workflow structured and easy to understand. The data folder contains the original datasets, the notebooks folder contains the Jupyter Notebook used for analysis, the charts folder stores generated visualisations, and the results folder contains exported tables and findings.

Some key findings from the analysis show that Brent Oil prices experience significant fluctuations across different periods, often influenced by supply disruptions and broader economic uncertainty. Gold displayed relatively more stable behaviour and is often viewed as a safer asset during uncertain times. The moving average analysis also helped smooth short-term fluctuations and better reveal long-term trends.

To run this project, first create a virtual environment, activate it and install the required libraries from the requirements file. Once the dependencies are installed, open Jupyter Notebook and run the notebook file available in the notebooks folder.

The analysis for this project was completed using Jupyter Notebook and Python libraries including pandas, NumPy and matplotlib. All required files, charts and result tables are included in the repository. The main notebook used for analysis can be found in the notebooks folder.
